# Infinite Adaptive Mario

Maven-ized version of Infinite Adaptive Mario.

## Quick start

	mvn package
	java -jar target/infinite-adaptive-mario-1.0.jar

Use the arrow keys to move and jump. Hold the space bar to run faster.
